Для початку роботи з програмою потрібно встановити інтерпретатор Python.
Зробити це можно з офіційного сайту https://www.python.org/downloads/ .
Виконайте встановлення за вказівками та відкрийте командну строку, перейдіть в папку SSC2019
введіть команду: pip install -r requirements.txt.
Після завершення встановлення необхідних пакетів,
запустіть файл SSC_main та ознайомтесь з інформацією про програму і керуванням
